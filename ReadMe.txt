Чтобы создать исполняемый файл:
https://habr.com/ru/companies/vdsina/articles/557316/

Если запускаем в первый раз:

1. Установка Auto PY to EXE
1.1 Установка через pip
pip install auto-py-to-exe
auto-py-to-exe --version

2. Запуск приложения
auto-py-to-exe
3. Для преобразования .py в .exe сначала необходимо указать путь к файлу Python. Достаточно перейти к местоположению файла, который нужно преобразовать, а затем добавить путь.
4. В интерфейсе есть опция выбора «One Directory» или «One File». Нас интересует «One Directory»
5. После этого нужно будет выбрать тип приложения: консольное (Console Based) или оконное (Window Based). Надо выбрать «Window Based»
6. Достаточно нажать на кнопку CONVERT .PY TO .EXE .
7. Подождать пока программа завершит сборку.
8. Появится папка output. В её корень нужно скопировать файл Lab1_optimal.ui
9. Теперь программа должна запускаться с помощью файла optimal_lab1.exe
